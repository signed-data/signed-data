## Overview

The `commodities.brazil` domain provides certified, signed data feeds for
Brazilian commodity markets. Sources include B3 futures contracts via Brapi,
physical crop prices from CONAB, and international benchmarks from the World
Bank.

## CONAB API limitation

CONAB's web service is not a formal public API. It is the backend of their web
interface. The ingestor uses defensive parsing with a `last_known_good`
fallback. If the endpoint structure changes, the ingestor fails gracefully and
logs an alert rather than emitting corrupted data.

## CEPEA exclusion

CEPEA / Esalq data is excluded because it requires scraping, which violates the
CDS principle: **no scraping**. CEPEA will be added if a proper API is published.
