## Visão geral

O domínio `commodities.brazil` fornece feeds de dados certificados e assinados para os
mercados brasileiros de commodities. As fontes incluem contratos futuros da B3 via Brapi,
preços físicos de safras da CONAB e benchmarks internacionais do Banco
Mundial.

## Fontes de dados

| Fonte | URI | Auth | Licença | Notas |
|---|---|---|---|---|
| Brapi (futuros B3) | `https://signed-data.org/sources/brapi.dev.v1` | nenhuma | MIT | Futuros em tempo real |
| CONAB | `https://signed-data.org/sources/conab.gov.br.v1` | nenhuma | público | API não oficial; parsing defensivo necessário |
| Banco Mundial | `https://signed-data.org/sources/api.worldbank.org.v2` | nenhuma | CC BY 4.0 | Benchmarks mensais |

## Limitação da API CONAB

O web service da CONAB não é uma API pública formal. É o backend da interface web
deles. O ingestor usa parsing defensivo com um fallback `last_known_good`.
Se a estrutura do endpoint mudar, o ingestor falha graciosamente e
emite um alerta em vez de produzir dados corrompidos.

## Exclusão da CEPEA

Dados da CEPEA / Esalq são excluídos porque exigem scraping, o que viola o
princípio do CDS: **sem scraping**. A CEPEA será adicionada se uma API apropriada for publicada.
