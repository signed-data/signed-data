## Jogos

### Mega Sena

- **Tipo de conteúdo:** `https://signed-data.org/vocab/lottery-brazil/mega-sena-result`
- **Números:** 6 de 1–60
- **Sorteios:** Quarta e Sábado
- **Endpoint:** `GET /portaldeloterias/api/megasena/{concurso}`

### Lotofácil

- **Tipo de conteúdo:** `https://signed-data.org/vocab/lottery-brazil/lotofacil-result`
- **Números:** 15 de 1–25
- **Sorteios:** Segunda–Sexta
- **Endpoint:** `GET /portaldeloterias/api/lotofacil/{concurso}`

### Quina

- **Tipo de conteúdo:** `https://signed-data.org/vocab/lottery-brazil/quina-result`
- **Números:** 5 de 1–80
- **Sorteios:** Segunda–Sexta
- **Endpoint:** `GET /portaldeloterias/api/quina/{concurso}`

### Lotomania

- **Tipo de conteúdo:** `https://signed-data.org/vocab/lottery-brazil/lotomania-result`
- **Números:** 20 de 100
- **Sorteios:** Segunda, Quarta, Sexta
- **Endpoint:** `GET /portaldeloterias/api/lotomania/{concurso}`

### Dupla Sena

- **Tipo de conteúdo:** `https://signed-data.org/vocab/lottery-brazil/dupla-sena-result`
- **Números:** Dois sorteios de 6 de 1–50
- **Sorteios:** Segunda, Sexta
- **Endpoint:** `GET /portaldeloterias/api/duplasena/{concurso}`

## Schema do payload (exemplo Mega Sena)

| Campo | Tipo | Descrição |
|---|---|---|
| `concurso` | `int` | Número do sorteio |
| `data_apuracao` | `string` | Data do sorteio (DD/MM/AAAA) |
| `data_apuracao_iso` | `string` | Data do sorteio (AAAA-MM-DD) |
| `local_sorteio` | `string` | Nome do local |
| `cidade_uf` | `string` | Cidade / estado |
| `dezenas` | `string[]` | Números sorteados, ordenados |
| `dezenas_ordem_sorteio` | `string[]` | Números na ordem do sorteio |
| `acumulado` | `boolean` | Verdadeiro se o prêmio acumulou |
| `premiacoes` | `PrizeTier[]` | Detalhamento do prêmio por faixa |
| `valor_arrecadado` | `number` | Receita total (BRL) |
| `valor_acumulado_proximo` | `number` | Estimativa do próximo prêmio (BRL) |
| `data_proximo_concurso` | `string` | Data do próximo sorteio |
