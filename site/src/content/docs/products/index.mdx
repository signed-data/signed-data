SignedData publishes a small set of focused MCP products that wrap public Brazilian
data behind cryptographically signed CDS envelopes. Every event is signed by
`signed-data.org` and verifiable independently with the [public key](https://signed-data.org/.well-known/cds-public-key.pem).

## How they fit together

All four products share the same architecture:

- **Standard**: every event is a [CDS v0.2.0](/docs/spec/v0-2-0/) JSON-LD envelope
- **Trust**: every event is RSA-PSS SHA-256 signed by `signed-data.org`; consumers verify with the public key
- **Transport**: Streamable HTTP MCP, deployed at `{product}.mcp.signed-data.org`
- **Code**: open-source in [`signed-data/cds`](https://github.com/signed-data/cds) under MIT

For the runtime architecture, see [Architecture](/docs/architecture/).
