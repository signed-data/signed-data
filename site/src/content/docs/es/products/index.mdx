SignedData publica un pequeño conjunto de productos MCP enfocados que envuelven datos públicos
brasileños detrás de envoltorios CDS firmados criptográficamente. Cada evento es firmado por
`signed-data.org` y verificable de forma independiente con la [clave pública](https://signed-data.org/.well-known/cds-public-key.pem).

## Cómo encajan entre sí

Los cuatro productos comparten la misma arquitectura:

- **Estándar**: cada evento es un envoltorio JSON-LD [CDS v0.2.0](/docs/spec/v0-2-0/)
- **Confianza**: cada evento está firmado con RSA-PSS SHA-256 por `signed-data.org`; los consumidores verifican con la clave pública
- **Transporte**: MCP HTTP Streamable, desplegado en `{product}.mcp.signed-data.org`
- **Código**: open source en [`signed-data/cds`](https://github.com/signed-data/cds) bajo MIT

Para la arquitectura en tiempo de ejecución, consulta [Arquitectura](/docs/architecture/).
