Datos firmados y verificados del registro de empresas brasileñas como herramientas MCP para Claude.
Perfiles CNPJ, registros de socios (QSA — *quadro societário*) y consultas de actividad económica
CNAE, todos provenientes de BrasilAPI (que replica la Receita Federal).

**Endpoint:** `https://companies.mcp.signed-data.org/mcp`
**Especificación del dominio:** [companies.brazil](/docs/spec/domains/companies-brazil/)

Este es un producto **dirigido por consultas** — no hay ingestión programada. Cada consulta de
CNPJ se obtiene y se firma en el momento de la solicitud. La firma CDS prueba *que* BrasilAPI
devolvió estos datos en la marca de tiempo de la consulta.

## Herramientas

| Herramienta | Descripción |
|---|---|
| `get_company_profile` | Perfil CNPJ completo — razão social, situação cadastral, endereço, capital social, CNAE |
| `get_company_partners` | Registros de socios / accionistas (QSA) de un CNPJ |
| `check_company_status` | Verificación rápida del estado `ATIVA` / `BAIXADA` |
| `validate_cnpj_tool` | Valida los dígitos verificadores del CNPJ sin hacer ninguna llamada a la API |
| `get_cnae_info` | Consulta un código de actividad económica (CNAE) |
| `batch_company_lookup` | Consulta hasta 10 CNPJs en una sola llamada |

## Validación de CNPJ

Todas las consultas validan los dígitos verificadores del CNPJ antes de hacer cualquier llamada a la API. Los CNPJs
inválidos se cortocircuitan con una respuesta de error y nunca llegan a BrasilAPI. Usa `validate_cnpj_tool`
para validar sin consumir una solicitud.

## Variables de entorno

| Variable | Descripción | Predeterminado |
|---|---|---|
| `CDS_PRIVATE_KEY_PATH` | Ruta a la clave privada RSA para firmar | (sin firma si no se establece) |
| `CDS_PUBLIC_KEY_PATH` | Ruta a la clave pública RSA para verificación | (omitir verificación) |
| `CDS_ISSUER` | URI del emisor en eventos firmados | `signed-data.org` |

<Aside type="caution">
La firma CDS atestigua *lo que devolvió BrasilAPI*, no el registro subyacente de la Receita
Federal en cualquier momento posterior. Trata los datos de las empresas como una instantánea puntual.
</Aside>
