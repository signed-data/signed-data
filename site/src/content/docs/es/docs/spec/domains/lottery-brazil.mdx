## Juegos

### Mega Sena

- **Tipo de contenido:** `https://signed-data.org/vocab/lottery-brazil/mega-sena-result`
- **Números:** 6 de 1–60
- **Sorteos:** miércoles y sábado
- **Endpoint:** `GET /portaldeloterias/api/megasena/{concurso}`

### Lotofácil

- **Tipo de contenido:** `https://signed-data.org/vocab/lottery-brazil/lotofacil-result`
- **Números:** 15 de 1–25
- **Sorteos:** lunes a viernes
- **Endpoint:** `GET /portaldeloterias/api/lotofacil/{concurso}`

### Quina

- **Tipo de contenido:** `https://signed-data.org/vocab/lottery-brazil/quina-result`
- **Números:** 5 de 1–80
- **Sorteos:** lunes a viernes
- **Endpoint:** `GET /portaldeloterias/api/quina/{concurso}`

### Lotomania

- **Tipo de contenido:** `https://signed-data.org/vocab/lottery-brazil/lotomania-result`
- **Números:** 20 de 100
- **Sorteos:** lunes, miércoles, viernes
- **Endpoint:** `GET /portaldeloterias/api/lotomania/{concurso}`

### Dupla Sena

- **Tipo de contenido:** `https://signed-data.org/vocab/lottery-brazil/dupla-sena-result`
- **Números:** Dos sorteos de 6 de 1–50
- **Sorteos:** lunes, viernes
- **Endpoint:** `GET /portaldeloterias/api/duplasena/{concurso}`

## Esquema del payload (ejemplo Mega Sena)

| Campo | Tipo | Descripción |
|---|---|---|
| `concurso` | `int` | Número del sorteo |
| `data_apuracao` | `string` | Fecha del sorteo (DD/MM/AAAA) |
| `data_apuracao_iso` | `string` | Fecha del sorteo (AAAA-MM-DD) |
| `local_sorteio` | `string` | Nombre del lugar |
| `cidade_uf` | `string` | Ciudad / estado |
| `dezenas` | `string[]` | Números sorteados, ordenados |
| `dezenas_ordem_sorteio` | `string[]` | Números en orden de sorteo |
| `acumulado` | `boolean` | True si el premio se acumuló |
| `premiacoes` | `PrizeTier[]` | Desglose de premios por nivel |
| `valor_arrecadado` | `number` | Recaudación total (BRL) |
| `valor_acumulado_proximo` | `number` | Estimación del próximo premio (BRL) |
| `data_proximo_concurso` | `string` | Fecha del próximo sorteo |
