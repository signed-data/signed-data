## Visión general

El dominio `commodities.brazil` proporciona feeds de datos certificados y firmados para los
mercados de commodities brasileños. Las fuentes incluyen contratos de futuros B3 vía Brapi,
precios físicos de cosechas de CONAB y benchmarks internacionales del Banco
Mundial.

## Fuentes de datos

| Fuente | URI | Auth | Licencia | Notas |

## Limitación de la API de CONAB

El servicio web de CONAB no es una API pública formal. Es el backend de su interfaz
web. El ingestor usa parseo defensivo con un fallback `last_known_good`.
Si la estructura del endpoint cambia, el ingestor falla con elegancia y
registra una alerta en lugar de emitir datos corruptos.

## Exclusión de CEPEA

Los datos de CEPEA / Esalq están excluidos porque requieren scraping, lo que viola el
principio CDS: **sin scraping**. CEPEA será añadido si se publica una API adecuada.
